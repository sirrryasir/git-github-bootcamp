# 🌿 Git & GitHub Bootcamp

Welcome to the **Git & GitHub Bootcamp** repository! 🚀

This repository contains all the lessons, exercises, assignments, and resources used during our **one-month Git & GitHub Bootcamp**.

The program is designed to take participants from **complete beginners to confident Git users**, teaching the workflows used by professional software engineers and open source contributors.

Throughout the bootcamp, you will learn how to:

1. Understand Version Control
2. Create and Manage Git Repositories
3. Track Changes with Git
4. Work with Branches
5. Merge Code and Resolve Conflicts
6. Collaborate Using GitHub
7. Create Pull Requests
8. Contribute to Open Source Projects

---

## 🎯 Goal

Our mission is simple:

Help Somali developers master one of the most important skills in modern software development.

By the end of this bootcamp, you won't just know Git commands—you'll understand how real developers collaborate, manage code, and contribute to projects used by people around the world.

---

📘 **Tip:**

Clone this repository, follow each lesson in order, complete the assignments, and practice every command on your own computer. Git is a practical skill—the more you use it, the more confident you'll become.
